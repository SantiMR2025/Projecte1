<h1 align="center">Proyecto SMX2 - Sistemas Microinformáticos y Redes</h1>

<hr>

<h2>1. Descripción</h2>
<p>
Este proyecto corresponde al CFGM de Sistemas Microinformáticos y Redes (2º curso).
Su objetivo principal es trabajar con una metodología profesional, simulando un entorno real de empresa.
</p>

<p>
El enfoque no se centra únicamente en el resultado técnico, sino en la organización,
la planificación y el trabajo en equipo.
</p>

<hr>

<h2>2. Objetivos</h2>

<ul>
<li>Aplicar metodología ágil (Kanban)</li>
<li>Organizar y planificar proyectos técnicos</li>
<li>Trabajar en equipo</li>
<li>Documentar procesos técnicos</li>
<li>Reforzar conocimientos de sistemas y redes</li>
</ul>
